## Baroque Sequences (for development sections)
| Pattern | Description | Usage |
|---|---|---|
| Descending 5ths: I-IV-vii-iii-vi-ii-V-I | Circle of fifths | 2-bar idea repeated down through the cycle |
| Descending 3rds + ascending step: C-Am-Dm-Bm... | Bass: down 3rd, up step | Baroque ground bass |
| Romanesca: III-VII-i-V | Renaissance ancestor of Pachelbel | Ground bass variation |
| Passamezzo: i-VII-i-V-III-VII-i-V-i | Renaissance/early Baroque | Extended ground bass |

## Cadence Quick Reference
| Ending Feel | Progression | When to Use |
|---|---|---|
| Strong finish | V-I | Section endings, song endings |
| Open / "to be continued" | any-V | Loop points, half-way points |
| Surprise deflection | V-vi | Extending before the real ending |
| Gentle close | IV-I | Peaceful sections, lullabies |
| Dramatic pause | iv6-V | Before boss entrances, key moments |
| Baroque minor ending | V-i with Picardy 3rd (V-I) | Raise the 3rd of final minor chord to major |

## Key/Mode Quick Reference for Game Contexts
| Context | Recommended Keys/Modes | Why |
|---|---|---|
| Adventure/wonder | Lydian (raised 4th) | Ethereal, floating quality |
| Epic/heroic | Major, Mixolydian | Bright but not fully resolved |
| Battle | Natural minor, Phrygian | Dark, aggressive |
| Mysterious | Dorian, whole-tone | Between major and minor |
| Exotic/ancient | Phrygian, harmonic minor | Half-step tension |
| Sad/reflective | Aeolian (natural minor) | Pure minor melancholy |
| Pastoral/calm | Pentatonic, Ionian (major) | Can't sound wrong |
| Sinister/evil | Locrian, diminished | Maximum instability |
